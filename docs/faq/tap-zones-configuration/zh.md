# 配置标签区域大小左/右/上/下

> 您可以配置选项卡区域(边缘)的大小和位置：

||||
|-|-|-|
|![](1.png)|

为了配置点击标签区域(短信区域)时的操作，您需要打开**设置**，
然后转到**高级功能**，找到**标签区域(短信区域)**部分，然后点击**设置**。

在所需选项卡区域对面的弹出窗口中，单击三角形并选择所需的操作。

安装后，不要忘记按**应用**

||||
|-|-|-|
|![](2.png)|![](3.png)|

要配置标签区域(短信区域)路由器，您需要打开**设置**。

然后转到**高级功能**，找到**标签区域(短信区域)**部分，然后选择标签区域路由器
