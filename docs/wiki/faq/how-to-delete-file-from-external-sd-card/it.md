# Come eliminare i libri dalla scheda SD esterna

Android 7+ non consente di eliminare i file dalla scheda SD esterna dal
ragioni di sicurezza.

* Scegli scheda SD
* Seleziona e consenti di eliminare dalla scheda SD scelta
